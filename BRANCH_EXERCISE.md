# ブランチ演習課題

## 演習 1: 機能ブランチの作成

1. `feature/contact-form` ブランチを作成してください
2. contact.html にフォームを追加してください
3. 変更をコミットしてください
4. main ブランチにマージしてください

## 演習 2: 複数ブランチでの作業

1. `feature/styling` ブランチでスタイルを改善
2. `feature/about-page` ブランチで自己紹介ページを充実
3. 両方のブランチを main にマージ

## 演習 3: コンフリクトの解決

1. 2 つのブランチで同じファイルの同じ箇所を編集
2. コンフリクトを発生させる
3. 適切に解決する
